# Project Overview

**Kemet Basket** is a **mobile app UI/UX design project** that showcases **Egyptian-made products** as alternatives to boycotted imports and promotes local goods across all categories.

<div align="center">
**تطبيق للمنتجات المصرية بديل المقاطعة** 🇪🇬
</div>

##  Core Focus
- **🇪🇬 Made in Egypt** — Food, clothing, home essentials  
- **📱 Mobile-first** design optimized for phones
- **🗣️ Bilingual** interface (English + Arabic)
- **✅ Verified** local sellers and products

##  Design Philosophy
Simple, intuitive, culturally-inspired design that makes supporting the Egyptian economy **easy and enjoyable**.
